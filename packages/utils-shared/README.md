# @fatcherjs/utils-shared

A package shared utils in fatcher

## Install

### NPM

```bash
>$ npm install @fatcherjs/utils-shared
```

### CDN

```html
<script src="https://cdn.jsdelivr.net/npm/@fatcherjs/utils-shared/dist/progress.min.js"></script>
```
